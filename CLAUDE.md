This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a personal resume repository hosted at https://github.com/Lision/Resume. It contains bilingual resume files in Markdown format.

## File Structure

- `RESUME.md` - English version of the resume
- `RESUME-ZH_CN.md` - Chinese (Simplified) version of the resume

## Content Maintenance

The two resume files are **parallel translations** of the same content. When updating the resume:

1. **Always update both files** - Changes to work experience, projects, or contact information should be reflected in both English and Chinese versions
2. **Maintain structural consistency** - Both files should have the same section hierarchy and organization
3. **Check for content parity** - Ensure bullet points, dates, and key details match between versions (accounting for translation differences)

## Commit Conventions

Based on repository history, commit messages typically use conventional commit format:
- `feat:` - New features or additions
- `remove:` - Removing content
- `--other:` - Miscellaneous updates

## No Build Process

This repository contains only Markdown documentation. There are no build commands, tests, linting, or dependencies to manage.
